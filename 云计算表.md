## 用户表t_user_info

USERNAME	用户名		varchar(50)

USERPWD	密码		varchar(16)

USERTYPE	用户类型 	int	0为管理员1为普通用户

STATUS		状态		int	0未登录1已登录

GOODSNAME	商品名称		varchar(100)

GOODSTYPE		商品类型		varchar(50)

GOODSDIS		商品描述		varchar(100)

GOODSNUM		商品库存		int

GOODSETC	 	其他		varchar(100)
